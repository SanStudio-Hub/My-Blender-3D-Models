### 📌 Overview

This project is a **3D model previewer** hosted on GitHub Pages.  
It allows users to upload `.glb` (GLTF binary) 3D model files — commonly exported from Blender or other 3D tools — and view them directly in the browser.

### 🌟 Features

- 📤 **Upload .glb files**
- 👁️ **View 3D models interactively**
- 🧰 **Clean, minimal interface**

### 💡 How to Use

1. Open the live link above.  
2. Click the **Upload** button.  
3. Choose a `.glb` file.  
4. Explore the 3D model on screen.

----

### 🧠 Ideal For

- Previewing Blender exports quickly without Blender installed
- Sharing 3D models with friends or clients
- Testing `.glb` models on web
